MathWorks platform offer the possibility to model physical systems using various approaches, from data-driven model using System Identification Toolbox or Deep Learning Toolbox up to representing system level equations using block diagram in Simulink. It is very common though to model multidomain system using a physical network approach based on acausal physical connection between components.
